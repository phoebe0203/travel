# AI旅游规划网页应用 - gooh旅记

## 项目简介

这是一个基于AI的智能旅游规划网页应用，参考了"gooh旅记"APP的设计风格。应用允许用户输入目的地、旅行日期、人数和预算，选择兴趣爱好，然后生成个性化的旅游计划。

## 功能特性

- **AI旅游规划**：根据用户输入的目的地、日期、人数、预算和兴趣爱好生成个性化旅游计划
- **响应式设计**：适配不同设备屏幕尺寸
- **现代化UI**：采用现代旅游应用设计风格，视觉效果美观
- **交互流畅**：平滑滚动、标签选择、加载动画等增强用户体验
- **热门目的地推荐**：展示热门旅游目的地，提供参考

## 如何使用

1. 打开 `index.html` 文件在浏览器中
2. 在首页点击"开始规划"按钮或导航到"AI规划" section
3. 填写以下信息：
   - 目的地：输入您想去的地方（如三亚、丽江、黄山等）
   - 旅行日期：选择开始和结束日期
   - 旅行人数：输入人数
   - 预算范围：选择经济型、舒适型或豪华型
   - 兴趣爱好：点击选择您感兴趣的活动类型（可多选）
4. 点击"生成旅游计划"按钮
5. 等待AI生成计划（约2秒）
6. 查看生成的详细旅游计划，按天展示活动安排

## 技术实现

- **前端框架**：纯HTML5 + CSS3 + JavaScript
- **响应式设计**：使用CSS Grid和Flexbox
- **交互效果**：原生JavaScript实现
- **图片资源**：使用Trae API生成旅游相关图片

## 支持的目的地

目前应用内置了以下目的地的详细活动信息：
- 三亚
- 丽江
- 黄山

对于其他目的地，应用会根据选择的兴趣爱好生成通用的旅游计划。
